ExamMate AI Architecture

Input

1. Notes PDF
2. Module PDF
3. Question Bank PDF

Processing

1. Extract text from Notes PDF
2. Extract text from Module PDF
3. Extract questions from Question Bank PDF

Retrieval

For each question:

1. Search uploaded notes
2. Search uploaded modules
3. Find matching content
4. Collect relevant paragraphs
5. Collect relevant diagrams

Answer Generation

Generate answer using:

- Retrieved content only
- University-style format
- Marks-based formatting
- Point-wise structure

Validation

Check:

- No unsupported information
- Proper headings
- Proper formatting
- Complete answer

Output

Generate downloadable Question + Answer PDF
